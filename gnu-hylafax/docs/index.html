Description

The gnu.hylafax package is a pure-Java implementation of the HylaFAX (FAX server) client protocol. The gnu.hylafax package can be used to implement a HylaFAX client that will run on any Java-supporting system.

The current release of this software includes an FTP client implementation in the gnu.inet.ftp package. We've worked to make this library compliant with the FTP RFC, RFC0959.

Technical Notes

The current implementation is intended to directly export the HylaFAX protocol with few "niceties." Therefore, the API does not implement things the Java way, yet. By the Java way, I mean proper getX and setX methods.

It is my intention to provide a more developer friendly API on top of the current API as development continues. I'm still formulating this higher-level API, so speak up if you'd like to offer suggestions.

The gnu.hylafax package is pretty useless without a working installation of the HylaFAX FAX server. The gnu.hylafax based client can be run on any Java supporting system with internet access to the HylaFAX server (local or remote.) For more information on HylaFAX, see the website.

For more information on planned features, check out the TODO file in the latest archive.

Installation

1. Download the latest version.
2. Use GNU tar (and gzip) to extract the files.
tar -xvzf gnu.hylafax-0.0.7.tar.gz

3. You will be left with the HTML-formatted documentation, a Java archive file containing the compressed bytecode for the gnu.hylafax package, and the complete library source as well as source to a number of related utilities.
4. Add the .jar file to your CLASSPATH. See the README file for help.
