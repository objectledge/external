XBIS XML Information Set Encoding

Welcome to the XBIS 0.9.5 release. The docs subdirectory contains the full JiBX documentation, mirroring the version posted on the web site as of the date of this release. This includes some information on running the main test programs included in the distribution, test.RunTest for performance testing (described on the Performance page) and test.RoundTrip (described on the Status page). There are also a set of Linux script files in this directory which can be used for running the performance tests.
